# Posts Directory

## Structure

- `fundamentals/` - Mon/Wed/Fri posts on AI & CS basics
- `papers/` - Tue/Thu/Sat posts on trending research papers

## Naming Convention

**Fundamentals:**
```
YYYY-MM-DD-topic-name.md
Example: 2025-11-11-attention-mechanism.md
```

**Papers:**
```
YYYY-MM-DD-paper-title-short.md
Example: 2025-11-12-chain-of-thought-reasoning.md
```

## Post Template

### Fundamentals
```markdown
---
layout: post
type: fundamental
title: "Attention Mechanism Basics"
date: 2025-11-11
topic: "Deep Learning"
keywords: ["attention", "transformer", "neural networks"]
---

[Content in 30-60 seconds read time]
```

### Papers
```markdown
---
layout: post
type: paper
title: "Chain-of-Thought Prompting"
date: 2025-11-12
category: "dialog"
arxiv_id: "2201.11903"
keywords: ["prompting", "reasoning", "LLM"]
---

[Paper summary in 1 minute]
```
