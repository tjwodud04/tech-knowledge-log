Backpropagation (Chain Rule for Training Neural Networks)

난이도 / Difficulty: Intermediate
날짜 / Date: 2026-07-20

내용

역전파는 연쇄법칙(chain rule)을 이용해 신경망 파라미터의 기울기를 계산하는 핵심 메커니즘으로, 대규모 모델을 빠르고 안정적으로 학습시키는 데 필수적입니다.

작동 원리: 순전파로 출력과 중간 활성화를 계산한 뒤, 손실에 대해 출력에서 입력 방향으로 미분을 전파(reverse-mode AD)합니다. 각 노드에서 연쇄법칙을 적용하여 로컬 그라디언트(출력에 대한 입력의 미분)를 곱해 전체 기울기를 얻습니다. 구현상으로는 계산 그래프를 구성해 노드별 로컬 그래디언트와 저장된 활성화를 이용해 역순으로 누적합니다.
실무 포인트: 대형 모델은 수백억 파라미터와 긴 연산 그래프를 가지므로 메모리/계산 절약 기법(그래디언트 체크포인팅, mixed-precision, 분산 옵티마이저—예: ZeRO)이 필수입니다. 또한 BPTT(Backpropagation Through Time)는 시퀀스 모델에서 시간 축으로 역전파하는 표준입니다.

활용

실제 코드(간단한 PyTorch 예):
import torch
import torch.nn as nn
import torch.nn.functional as F

model = nn.Linear(10,1).cuda()
opt = torch.optim.Adam(model.parameters())
scaler = torch.cuda.amp.GradScaler()
accum_steps = 4
for i, (x,y) in enumerate(loader):
x,y = x.cuda(), y.cuda()
with torch.cuda.amp.autocast():
pred = model(x)
loss = F.mse_loss(pred, y) / accum_steps
scaler.scale(loss).backward()
torch.nn.utils.clip_grad_norm_(model.parameters(), 1.0)
if (i+1) % accum_steps == 0:
scaler.step(opt); scaler.update(); opt.zero_grad()

커스텀 연쇄법칙(autograd.Function):
class Square(torch.autograd.Function):
@staticmethod
def forward(ctx, x):
ctx.save_for_backward(x)
return x*x
@staticmethod
def backward(ctx, grad_output):
x, = ctx.saved_tensors
return grad_output * 2 * x

x = torch.tensor(2.0, requires_grad=True)
y = Square.apply(x); y.backward(); print(x.grad)

사용 위치: PyTorch(메타/페이스북 등)와 TensorFlow(구글)는 역전파 기반 자동미분을 핵심으로 사용합니다. 대규모 트레이닝에서는 Microsoft DeepSpeed, NVIDIA의 mixed-precision/CuDNN 최적화, Google의 JAX/XLA(연구·프로덕션)가 흔히 채택됩니다. 연구 참고: Rumelhart et al. 1986, Werbos 1974(초기 제안). Best practice: gradient clipping, fp16 + AMP, 체크포인팅, 분산 옵티마이저(ZeRO) 결합.

Content

Backpropagation uses the chain rule to compute gradients for network parameters; it's the fundamental algorithm enabling efficient training of deep models in production.

How it works: forward pass computes activations and loss; reverse-mode automatic differentiation (AD) walks the computation graph backwards, applying the chain rule at each node to accumulate gradients wrt parameters. Each node computes a local derivative and multiplies incoming gradient signals, so complexity scales with graph edges rather than parameter count per se. For temporal models, Backpropagation Through Time (BPTT) unrolls the recurrence and applies the same reverse pass across time steps.
Practical considerations: large-scale training requires memory/compute optimizations—gradient checkpointing (trade compute for memory), mixed-precision (FP16 + loss scaling), gradient clipping, and distributed optimizers (e.g., ZeRO) to shard optimizer state and gradients. Autodiff implementations are in PyTorch (used widely in industry), TensorFlow, and JAX (Google Research).
